# Debian GUI on Termux

Debian proot-distro with Termux:X11 XFCE desktop.

## Scripts

Start GUI:
./scripts/start-debian-gui.sh

Stop GUI:
./scripts/stop-debian-gui.sh

Root terminal:
./scripts/debian-root.sh

User terminal:
./scripts/debian-suba.sh

## Features

- Debian Linux on Android
- XFCE Desktop
- Termux:X11
- Separate root and normal user access
- Sudo configured
