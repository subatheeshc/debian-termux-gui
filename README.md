# Debian Termux GUI Setup

A simple automated setup project to run Debian with GUI environment on Android using Termux, proot-distro and Termux:X11.

## Features

- Automated Debian installation
- XFCE desktop environment setup
- Termux:X11 GUI support
- Root and normal user login support
- Sudo permission configuration
- Launcher scripts for easy start and stop
- Dynamic Debian username support

## Requirements

- Android device
- Termux (latest version recommended)
- Internet connection
- Storage space for Debian and GUI packages
